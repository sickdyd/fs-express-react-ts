# What is it?

This is a template project to create an application with the following structure:

```
Backend: Node.js + Express
Frontend: ReactJS
```

# Backend

Run api for development (nodemon):

```
yarn api:dev
```

Run the api for production:

```
yarn api:start
```

Create api build:

```
yarn api:build
```

Start the debugger for Chrome:

```
yarn api:debug
```

Run all checks:

```
yarn api:checks
```
